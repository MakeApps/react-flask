#Using Python Flask JWT Token and ReactJS.

Clone the Repo
- git clone git@github.com:MakeApps/react-flask.git
- cd react-flask
- pip install flask
- pipenv shell

Run the Flask Server 
- FLASK_APP=app.py FLASK_ENV=development flask run

#Front End 
Install npm packages.
- npm install
- npm run watch
